React/Vite + Tailwind app with localStorage and .ics calendar export.

## Setup

```bash
# 1. Unzip and enter folder
cd divine-protocol-app

# 2. Install dependencies
npm install

# 3. Run locally
npm run dev

# 4. Open http://localhost:5173
```

## Build for Production

```bash
npm run build
```

Output goes to `/dist` folder. Host anywhere that serves static files.

## Features

- Auto-adjusting daily schedule based on wake time
- .ics export for phone calendar notifications
- Meal builder with fruit pairing rules
- Weekly meal planning
- Grocery list generator
- All data in localStorage
- PWA-ready (Add to Home Screen)
